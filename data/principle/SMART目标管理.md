## 🧭 一、什么是 SMART 原则？

SMART 是五个英文单词首字母的缩写，代表：

| **英文**             | **中文含义** | **说明**            |
| ------------------ | -------- | ----------------- |
| **S - Specific**   | 明确具体     | 目标必须清晰、聚焦，而不是模糊泛泛 |
| **M - Measurable** | 可衡量      | 有量化标准或可感知的结果      |
| **A - Achievable** | 可实现      | 目标现实、有资源支持        |
| **R - Relevant**   | 有相关性     | 与你的核心目标或角色相关      |
| **T - Time-bound** | 有时限      | 有清晰的截止时间或阶段进度     |

## 🛠️ 二、如何用 SMART 原则写一个好目标？

比如你想优化“阅读计划”这个目标，普通写法可能是：

> 我要多读书。

用 SMART 原则改写后：

> 我将在未来一个月内，每周读完 1 本关于时间管理的书（如《搞定》或《番茄工作法》），并写 100 字以上的笔记。

|   |   |
|---|---|
|SMART 维度|对应内容|
|S|明确是“时间管理的书”|
|M|每周 1 本、写笔记|
|A|1 本/周对你来说是合理的|
|R|与你正在搭建时间管理体系高度相关|
|T|限定在“未来一个月”内|

## 🧩 三、将 SMART 结合现有的时间管理系统

### ✅ 1. GTD 中「项目」和「下一步行动」用 SMART 定义目标

- GTD 项目：用 SMART 写清楚要完成什么、何时完成 → 见 [[GTD工作法框架]]、[[项目模板]]
- 下一步行动（Next Action）：要够具体，符合 SMART 中的「S」→ 见 [[全局任务池]]

### ✅ 2. 在 ABC 分类中，判断是否是「高价值任务」

SMART 目标通常都是 **A 类任务**，因为它们清晰、重要、推进核心目标 → 见 [[ABC + 4D工作法判断优先级]]

### ✅ 3. 4D 工作法中判断是否「值得自己做」

不符合 SMART 的模糊任务常常应该被 **删除（Delete）或委托（Delegate）**；符合 SMART 的任务，适合 **自己做（Do）或延后（Defer）** → 见 [[决策与执行速查]] 第四节

## 🧪 四、练习：帮你优化一个目标

- 🎯 我的任务目标是：________
    
- ✅ 具体（S）：它指的是什么？
    
- 📏 可衡量（M）：怎么知道完成了？
    
- 🚀 可实现（A）：我有资源/时间支持吗？
    
- 🎯 相关性（R）：它和我现在的目标有什么关系？
    
- ⏳ 时限（T）：什么时候完成？
